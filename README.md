# Autonomous Research Agent (RAG + MCP)

An end-to-end AI research agent that ingests unstructured documents,
performs retrieval-augmented generation (RAG), and generates
citation-grounded reports using cloud-hosted LLMs.

## Tech Stack
- LLM: Groq (LLaMA 3)
- RAG: FAISS + SentenceTransformers
- Agent Context: MCP
- Backend: FastAPI
- UI: Streamlit
